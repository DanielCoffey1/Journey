Journey Companion

A from-scratch Hyprland desktop for vanilla Arch Linux. One shell command and you're configured.

curl -fsSL https://get.journey.dev | bash

What you get

  • Hyprland tiling WM with sensible binds (SUPER+SPACE launcher, SUPER+ALT+SPACE menu).
  • Walker-based Journey Companion launcher.
  • Waybar, mako, SwayOSD, hyprlock, hypridle — pre-themed and wired together.
  • One journey CLI for everything: journey theme set, journey menu system, journey update, …
  • Live theme switching across walker, waybar, mako, swayosd, alacritty, ghostty, foot, btop.
  • Auto-detected hardware drivers (NVIDIA, Intel, T2 MacBook, Surface, Dell XPS, Framework 16, ASUS).

Requirements

  • Vanilla Arch Linux, booted to a TTY (no display manager needed for install).
  • An unprivileged user with sudo access.

After install

journey help           # list every subcommand
journey theme list     # see installed themes
journey update         # pull Journey + sync packages
journey uninstall      # clean removal (restores backups)
